# 🏎️ Honda Service Admin Panel

A robust and efficient administration dashboard built with **Flutter** and **Firebase** for managing automotive service records. 

---

## 🚀 Key Features
* **📊 Smart History Tracking:** Real-time synchronization with Firestore to browse all service records.
* **📅 Date-Specific Filtering:** Built-in calendar picker to instantly view reports for a specific day.
* **🔍 Live Search:** Instant filtering by car model or license plate number.
* **📄 PDF Report Generation:** Automatic generation of professional, print-ready service reports for customers.
* **💻 Cross-Platform Support:** Optimized for **Windows Desktop** (primary target), macOS, and Android.

---

## 🛠 Tech Stack
* **Framework:** [Flutter](https://flutter.dev) (Desktop & Mobile)
* **Backend:** [Firebase Cloud Firestore](https://firebase.google.com)
* **Language:** Dart
* **Major Packages:** `pdf`, `printing`, `firebase_core`, `cloud_firestore`
